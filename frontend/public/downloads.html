Aplicativo Principal

Felipinho Launcher

Essencial

O Launcher é o coração do sistema. Ele faz login com sua conta, monitora suas viagens em tempo real e envia os dados de telemetria automaticamente para o painel da transportadora.

Sistema Windows 10/11 (64-bit)
Versão v2.0 · Atualizado
Segurança Sem vírus · Código aberto
Baixar Launcher (.exe)
Importante: o Launcher só registra quando você está no caminhão
O Launcher não funciona no menu do jogo, na tela de carregamento ou no mapa. A telemetria só é enviada enquanto você estiver dentro do caminhão. Sempre abra o ETS2 pelo próprio Launcher e aceite a carga depois de estar logado.
Como instalar
1
Baixe o arquivo setup_GR_Expresso.exe acima
2
Execute como administrador e siga o instalador
3
Faça login com sua conta do painel Felipinho Launcher
4
Inicie o ETS2 pelo botão do Launcher — pronto!
Plugin do Jogo

SDK Telemetria ETS2

Necessário

Plugin oficial scs-telemetry que conecta o Euro Truck Simulator 2 ao Launcher. Sem ele, a telemetria não funciona e suas viagens não são registradas.

Jogo Euro Truck Simulator 2
Arquivo scs-telemetry.dll
Destino /bin/win_x64/plugins/
Baixar Plugin (GitHub)
Após baixar, copie o arquivo scs-telemetry.dll para a pasta plugins dentro do diretório de instalação do ETS2.
Como instalar
1
Acesse o GitHub e baixe o scs-telemetry.dll mais recente
2
Navegue até: Pasta Que Esta o Jogo\Euro Truck Simulator 2\bin\x64\plugins\
3
Cole o arquivo .dll nesta pasta (crie-a se não existir)
4
Reinicie o ETS2 e o Launcher detectará automaticamente
